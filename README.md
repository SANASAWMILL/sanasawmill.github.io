## How to Update Content

To update text, phone numbers, services, etc., edit `index.html` directly through GitHub's web interface:

1. Click the file `index.html` in this repo
2. Click the pencil icon (Edit) in the top right
3. Find the text you want to change
4. Make your edit
5. Scroll down, click "Commit changes"
6. Site updates within 1–2 minutes

To update photos:
1. Open the `images/` folder
2. Upload a new file with the same name to replace, OR
3. Upload a new file and update the reference in `index.html`
